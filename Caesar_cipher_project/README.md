# Caesar Cipher Project

## What this project is
This project is a Python implementation of the classic Caesar cipher. It allows users to encrypt (encode) and decrypt (decode) messages by shifting the alphabet by a user-defined number of positions.

## What problem it solves
It provides a simple way to obscure sensitive text messages from casual readers. It serves as an introductory tool to understand the basics of cryptography and text manipulation.

## What tech stack it uses
- **Language:** Python 3
- **Modules:** Custom `art` module for ASCII logos.

## Honest limitations
- **Security:** The Caesar cipher is an ancient encryption technique and is highly vulnerable to brute-force and frequency analysis attacks. It should never be used for actual secure data.
- **Character Support:** It only shifts standard lowercase alphabetic characters. Numbers, symbols, and uppercase characters are left unmodified but are not actively shifted.
- **Console-based:** Operates purely through standard terminal input and output.
