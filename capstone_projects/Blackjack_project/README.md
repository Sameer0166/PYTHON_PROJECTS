# Blackjack Project

## What this project is
This project is a terminal-based simulation of the popular casino card game, Blackjack. Players are pitted against a computer dealer to get a hand value as close to 21 as possible without busting.

## What problem it solves
It provides an accessible way to play and practice Blackjack strategies locally without needing physical cards or an internet connection.

## What tech stack it uses
- **Language:** Python 3
- **Modules:** Relies on the built-in `random` module for drawing cards and custom modules for logic and ASCII art.

## Honest limitations
- **Basic Mechanics:** It only simulates the core gameplay (hitting, standing). It may lack advanced casino rules like splitting pairs, doubling down, or insurance.
- **Console Interface:** Purely command-line based with ASCII visual elements rather than a modern graphical interface.
- **State Persistence:** Betting systems or prolonged tracking of winnings across multiple play sessions might not be robustly supported.
