# Python Beginner Projects Collection

## What this project is
This is a collection of beginner-friendly Python scripts located in the root directory. It includes:
- **Password Generator:** A tool that generates random and secure passwords based on user preferences.
- **Tip Calculator:** A utility to easily split a bill and calculate the tip for a group.
- **Treasure Game:** A text-based interactive adventure game where players make choices to find the hidden treasure.
- **Rock Paper Scissors:** A classic console-based rock, paper, scissors game against a computer opponent.

## What problem it solves
These scripts serve as foundational practice for Python programming. They solve simple, everyday problems (like calculating tips or generating passwords) and offer basic interactive entertainment, helping to build logic and understand control flow, loops, and basic input/output in Python.

## What tech stack it uses
- **Language:** Python 3 (Pure Python)
- **Libraries:** Built-in `random` module.

## Honest limitations
- **User Interface:** All scripts are strictly terminal/console-based without any Graphical User Interface (GUI).
- **Error Handling:** They assume ideal user input and may crash if invalid data types are entered (e.g., entering text when a number is expected).
- **Complexity:** The logic is linear and mostly relies on basic `if-else` branching and simple loops.
- **Security:** The password generator uses the standard `random` module, which is suitable for basic use but not cryptographically secure.
