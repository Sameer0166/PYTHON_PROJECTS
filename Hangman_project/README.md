# Hangman Project

## What this project is
This is a fully playable, text-based Hangman game. The computer randomly selects a word, and the player must guess it letter by letter before running out of lives (represented by drawing a hangman figure).

## What problem it solves
It solves the problem of needing a quick, entertaining mental exercise or word puzzle. From a development standpoint, it demonstrates string manipulation, list handling, and state management in Python.

## What tech stack it uses
- **Language:** Python 3
- **Modules:** Built-in `random` module, alongside custom `hangman_art` and `hangman_words` files to manage assets.

## Honest limitations
- **Static Word List:** The game relies on a hardcoded list of words rather than pulling dynamically from an API or a larger dictionary file.
- **Text Interface:** Relies entirely on ASCII art for the visual representation of the hangman.
- **Input Validation:** Limited validation on user inputs; entering multiple characters at once or special characters might lead to unintended behavior.
