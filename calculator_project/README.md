# Command Line Calculator

## What this project is
A functional, terminal-based calculator capable of performing standard arithmetic operations like addition, subtraction, multiplication, division, exponents, and modulus. It allows chaining operations sequentially.

## What problem it solves
It solves the need for performing quick and continuous mathematical calculations directly from the command line without opening a heavy graphical application.

## What tech stack it uses
- **Language:** Python 3
- **Concepts:** Uses dictionaries to map operator strings to specific calculation functions.

## Honest limitations
- **Order of Operations:** Because it calculates sequentially step-by-step with user input, it does not naturally support standard mathematical order of operations (BODMAS/PEMDAS) for long equations entered at once.
- **Error Handling:** While it prevents division by zero gracefully, inputting non-numeric values for numbers will crash the script.
- **Console Interface:** Purely text-based, requiring keyboard input for every step.
