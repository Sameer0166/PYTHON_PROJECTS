# Number Guessing Project

## What this project is
A fun, interactive console game where the program randomly selects a number between 1 and 100, and the player tries to guess it. The game gives hints if the guess is "Too High" or "Too Low".

## What problem it solves
It provides a simple logic and deduction game for entertainment. It helps players practice binary search strategies mentally and serves as a great introductory project for managing loops, functions, and game states in Python.

## What tech stack it uses
- **Language:** Python 3
- **Modules:** Built-in `random.randint` module.

## Honest limitations
- **Replayability:** The user must manually run the script again after finishing a game; it lacks a loop to seamlessly restart without exiting.
- **Error Handling:** Entering a non-integer value when prompted for a guess will cause the program to crash with a `ValueError`.
- **Difficulty Scaling:** The difficulty is strictly limited to "easy" (10 turns) or "hard" (5 turns) with no custom options.
